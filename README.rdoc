To play 2048 game in your terminal follow the following procedure

1. Clone this repo

2. cd short_url in your terminal

3. Type ruby game.rb in your terminal

4. Use "a=>left, s=>down, d=>right, w=>up"

5. Type the key and press enter until the game gets end
